Utilizando o dataset IRIS para criar um classificador . 
Nessa atual versão, na coluna variety, substituímos as classificações "setosa", "versicolor" e "virgínica" por 1,2,3 respectivamente para facilitar o desenvolvimento do programa.

Foram utilizados alguns tipos de plot para melhor visualização dos dados. histograma, matriz de correlação, gráfico 3d são exemplos desses.
